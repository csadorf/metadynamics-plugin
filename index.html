metadynamics Documentation

Welcome to the documentation for the HOOMD-blue metadynamics plugin !

Metadynamics is a technique to measure free energy surfaces (FES) using an adaptive bias potential.

Usage:

To load the plugin in HOOMD-blue, use

from hoomd_plugins import metadynamics

anywhere in your script file before using one of the below commands.
